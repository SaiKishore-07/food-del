# 🍔 Food Delivery Web Application

A full-stack Food Delivery Web Application built using **React, Node.js, Express, and MongoDB**.
This project includes **User Panel, Admin Panel, and Backend API** to manage food items, orders, and users.

---

## 🚀 Features

* User can browse food items
* Add to cart functionality
* Place order system
* Admin can add / update / delete food items
* Order management
* MongoDB database integration
* REST API backend
* Responsive UI

## ⚙️ Installation & Setup

1. Clone the repository

```
git clone https://github.com/SaiKishore-07/food-del.git
```

2. Install frontend

```
cd frontend
npm install
npm start
```

3. Install backend

```
cd backend
npm install
npm start
```

4. Install admin

```
cd admin
npm install
npm start
```
